# Currency Converter

## 📌 Project Overview

The Currency Converter is a Java console-based application developed as part of the **Java Programming Internship at Decode Labs**.

The application allows users to convert an amount from one currency to another using predefined exchange rates through a simple and interactive menu-driven interface.

---

## 🎯 Objective

Develop a Java application that:

- Accepts source currency
- Accepts destination currency
- Accepts amount
- Converts the currency
- Displays the converted amount
- Performs input validation

---

## ✨ Features

- Convert between INR, USD, EUR and GBP
- Menu-driven interface
- Input validation
- Accurate currency conversion
- Continuous conversion until exit
- Easy-to-use console application

## ▶️ How to Run

1. Open the project in IntelliJ IDEA.
2. Configure JDK 17 or above.
3. Run Main.java.
4. Select the source currency.
5. Select the destination currency.
6. Enter the amount.
7. View the converted amount.
8. Repeat or exit.

---

## 📝 Algorithm

1. Start the program.
2. Display available currencies.
3. Read source currency.
4. Read destination currency.
5. Read amount.
6. Validate the amount.
7. Convert the source currency into INR.
8. Convert INR into the target currency.
9. Display the converted amount.
10. Repeat until Exit is selected.
11. End the program.

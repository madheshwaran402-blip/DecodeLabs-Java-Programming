# Student Grade Calculator

## Project Overview

The Student Grade Calculator is a Java console application developed as part of the **Java Programming Internship at Decode Labs**.

The application accepts marks for multiple subjects, calculates the total marks and average percentage, assigns a grade, and displays the student's overall result.

---

## Features

- Student Name Input
- Multiple Subject Marks
- Input Validation (0–100)
- Total Marks Calculation
- Average Percentage Calculation
- Grade Assignment
- Pass/Fail Status
- Console-Based Interface

## How to Run

1. Open the project in IntelliJ IDEA.
2. Configure JDK 17 or above.
3. Open Main.java.
4. Click Run.
5. Enter the student name.
6. Enter the number of subjects.
7. Enter marks for each subject.
8. View the generated report.

## Grade Criteria

| Percentage | Grade |
|------------|-------|
| 90 - 100 | A |
| 80 - 89 | B |
| 70 - 79 | C |
| 60 - 69 | D |
| Below 60 | F |

---

## Algorithm

1. Start Program
2. Read Student Name
3. Read Number of Subjects
4. Accept Marks for Each Subject
5. Validate Marks
6. Calculate Total Marks
7. Calculate Average Percentage
8. Determine Grade
9. Determine Pass/Fail Status
10. Display Student Report
11. End Program
